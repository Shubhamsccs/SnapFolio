# Snapfolio: Moments. Magnified. 📷

A clean, visually focused landing page designed for a photography portfolio. I built this project during my early stages of learning web development to practice structuring web pages and styling with CSS.

## 🚀 Tech Stack
* **HTML5:** Semantic structure and layout.
* **CSS3:** Styling, colors, and typography.

## ✨ Features
* Attractive header and navigation layout.
* Hero section focused on visual storytelling.
* Built entirely without frameworks to build a strong fundamental understanding of CSS.

## ⚠️ Known Limitations
* **Not Responsive:** This project was built to practice basic layouts and is strictly optimized for **desktop viewing**. It is not yet optimized for mobile or tablet screens.

## 🛠️ How to View
Since this is a static website, you don't need any special server to run it.
1. Clone this repository or download the ZIP file.
2. Open the folder on your computer.
3. Double-click the `SnapFolio.html` (or `index.html`) file to open it in any web browser.
